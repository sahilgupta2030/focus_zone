# Collaborative Workspace
A Trello-like app with real-time collaboration, drag-and-drop boards/lists/cards, file attachments, activity log and multi-user roles.

## Features
- Real-time sync with Socket.IO
- Boards, lists, cards with drag-and-drop
- Attachments (S3/GridFS)
- Activity feed, dark mode, mobile responsive
